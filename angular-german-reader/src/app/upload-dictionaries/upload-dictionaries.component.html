My Dictionaries

Recommendation for Dictionary Use: Due to copyright restrictions, the application can only include one dictionary, which is the Wiktionary. However, you can upload your other dictionary data as well, such as the data available on dict.cc.

For bigger files, consider using a client-side text file splitter like this.

The uploaded dictionary must be a tab-separated text file in UTF-8 format with at least two columns: word and definition. If the file has more than two columns, the remaining columns will be ignored.

You can upload multiple dictionaries by refreshing the page between uploads.
